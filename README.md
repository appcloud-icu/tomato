# tomato

[![Deploy to GitHub Pages](https://github.com/appcloud-icu/tomato/actions/workflows/deploy.yml/badge.svg)](https://github.com/appcloud-icu/tomato/actions/workflows/deploy.yml)

A minimal Pomodoro timer application built with TypeScript and Vite.

## Getting Started

```bash
# Install dependencies
bun install

# Start development server
bun run dev

# Build for production
bun run build

# Preview production build
bun run preview

# Format code
bun run format
```
